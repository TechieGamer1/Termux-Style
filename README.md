## How To Install

Follow the steps below - 

```bash
apt update && yes | apt upgrade && apt update && apt install git -y
git clone https://github.com/TechieGamer1/Termux-Style
cd Termux-Style
./install
termux-style
```

### Features

+ 90 popular color-schemes.
+ 20 powerline patched fonts.
+ Randomly change color-schemes.
+ Import color-schemes from *local file* or *file URL*.
+ Set colors and fonts in place.
